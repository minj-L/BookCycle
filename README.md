## Development   
* HTML
* JavaScript
* JQuery 3.6.0   
* Apache Tomcat v8.5

### ✉️ 프로젝트 한 줄 소개  
`HTML`과 `JavaScript`, `Jquery`를 이용하여 만든
중고책 판매, 나눔 사이트 입니다. 프론트 엔드 프로젝트 입니다.

### Commit Message Convention   

```
Feat : 새로운 기능 추가
Fix : 버그 수정
Docs : 문서 수정
Style : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
Refactor : 코드 리펙토링
Chore : 기타 작업
Branch 이름은 기능별로 생성
```
